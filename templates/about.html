About CafeBuddy

It is a sophisticated Hotel recommendation system and reviews classification on Hybrid model using Machine Learning.

It let's you find new hotels around you. With this app you can find places you like, and discover new points of interest wherever you are.

Why hybrid?

We have used 3 different approaches of collaborative filtering to recommend hotels that fit user needs.

1- Recommend hotels based on their Popularity

2- Find similar hotels using K-Nearest Neighbours algo to find matching hotels based on user interests

3- Filter hotels using semantic analysis of reviews based on opinion mining technique
